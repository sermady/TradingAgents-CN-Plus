### 2️⃣ 数据质量检查报告

**文件**: `tradingagents/dataflows/optimized_china_data.py`

**新增方法**: `_check_data_quality` (约第1115行)

**功能描述**:
- 检查关键字段是否缺失（行业、营收增速、净利润增速、核心财务指标）
- 计算数据质量评分（满分100分）
- 生成质量等级（优秀/良好/一般/较差）
- 提供改进建议

**评分规则**:
- 每个关键字段缺失扣15分
- 质量等级:
  - 90-100分: 优秀
  - 70-89分: 良好
  - 50-69分: 一般
  - <50分: 较差

**报告示例**:
```markdown
## 📋 数据质量报告
- **质量评分**: 55/100 (一般)
- **缺失字段**: 所属行业, 营收同比增速, 净利润同比增速
- **改进建议**:
  - 行业信息缺失，可能影响行业对比分析
  - 营收同比增速数据缺失，无法评估营收增长情况
  - 净利润同比增速数据缺失，无法评估盈利能力变化
```

### 3️⃣ 自动触发数据同步

**文件**: `tradingagents/dataflows/optimized_china_data.py`

**新增方法**: `_trigger_background_sync` (约第1161行)

**功能描述**:
- 当检测到关键数据缺失时，自动触发后台数据同步

- 异步执行，不影响当前分析流程

**触发条件**:
- 所属行业缺失
- 营收同比增速缺失
- 净利润同比增速缺失

**执行流程**:
```python
if 有关键字段缺失:
    在后台线程/任务中:
        从Tushare获取最新数据

        记录同步日志
```
